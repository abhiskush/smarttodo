# Change Log

All notable changes to the "SmartTodo" extension will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-21

### Added
- ✨ **Core Todo Management**: Create, read, update, and delete todos
- 🎨 **Priority Levels**: Assign priorities (Low, Medium, High, Urgent, Critical) to tasks
- 🎯 **Visual Priority Indicators**: Color-coded priority levels for quick identification
- 🔍 **Advanced Filtering**: 
  - Filter by title (partial matches, case-insensitive)
  - Filter by completion status
  - Filter by priority level
  - Multi-field filtering support
  - Real-time search
- 💾 **Data Persistence**: Todos saved locally and persist across VS Code sessions
- 🎭 **Theme Support**: Full support for VS Code Light and Dark themes
- 📱 **Responsive Design**: Optimized for various screen sizes and resolutions
- 🎪 **Intuitive UI**: Clean, professional interface integrated into VS Code
- 📝 **Todo Details**:
  - Title (editable)
  - Completion status (toggle via checkbox)
  - Priority level (selectable from dropdown)
  - Visual distinction for completed todos (faded appearance)

### Features
- Command: `SmartTodo: Open Todo List` - Opens the todo management panel
- Webview-based UI for seamless integration with VS Code
- Keyboard navigation support
- No external dependencies required
- Lightweight and performant
